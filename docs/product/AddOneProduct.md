**Add one product**
----
Post a json data about one new product

* **URL**

  <_/api/products_>

* **Method:**

  `POST` 

* **Data Params**

  ```json
  OneProduct without id field
  ```
  
* **Success Response:**

  * **Code:** 200 <br />
    **Content:** 
    ```json
    {
        "status": "success",
        "id": "3"
    }
    ```
