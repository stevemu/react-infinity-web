**Show one product**
----
Returns json data about a single product

* **URL**

  <_/api/products/:id_>

* **Method:**

  `GET` 

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** 
    ```json
      {
        "id": 1,
        "model": "Evoke",
        "price": 4000,
        "profileImage": "Evoke_black_1.png"
      }
    
    ```
